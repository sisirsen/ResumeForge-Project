# ResumeForge

ResumeForge is a modern web application that helps users create professional, clean, and job-ready resumes.

# Goal
My goal behind this project to help students also professionals to make good quality resumes for their work/job purpose, so they dont have to think about rejection for the resumes, and can grow their other necessary skills to get jobs/internships.

## Features

1. Dynamic Resume Form Input.
2. Clean and Responsive UI.
3. Real-time Resume Preview.
4. Authentication System.@ Download Resume.
5. Fully browser-based (no installation required).
